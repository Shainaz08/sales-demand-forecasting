# 📈 Sales & Demand Forecasting — ML Task 1

A machine learning project that forecasts store sales using time series analysis and Random Forest.

## 🛠️ Tech Stack
- Python, Jupyter Notebook
- Pandas, Matplotlib
- Scikit-learn (Random Forest)
- Time series feature engineering

## 📊 Features
- ✅ Total sales trend analysis (2013–2017)
- ✅ Actual vs Predicted sales comparison
- ✅ 30-day future sales forecast
- ✅ Feature importance analysis
